## Core Features & Services

### Photo Geotagging Tools
1. **Add GPS to Photos Online**
   - Upload photos and add GPS coordinates
   - Manual coordinate entry or map selection
   - Batch processing support
   - Real-time preview

2. **Remove Geotag from Photos**
   - Strip GPS data from photos
   - Preserve other EXIF data
   - Privacy protection
   - Batch removal capability

3. **Edit EXIF Data**
   - Comprehensive EXIF data editor
   - Modify camera settings, date, time
   - Add custom metadata
   - Validate data integrity

### Technical Features
- **Client-side Processing**: All photo processing happens in browser
- **Privacy Focused**: No photos uploaded to servers
- **Mobile Responsive**: Works on all devices
- **Fast Processing**: Immediate results
- **No Registration Required**: Free to use without signup
