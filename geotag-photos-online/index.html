Geotag a Whole Set of Photos in One Pass

This page is built for the case where you have several photos from the same place — a hike, a wedding, a real-estate shoot, an event — and the camera didn't record GPS. Drop them all in, pick the location once on the map, and the same coordinates are written into every file's EXIF GPS block.

Each file is processed in your browser, one after the other, and packaged for download. There is no upload to a server, so the speed is limited only by your own machine. A typical batch of 10–20 JPEGs from a phone or DSLR finishes in a few seconds.

When this is the right tool

  • Trip photos with no GPS — older cameras, mirrorless bodies without GPS, or phone shots taken with location services off. Add location data to your images effortlessly.
  • Scanned film and prints — digitised analog photos have no EXIF GPS at all; you can backfill the location.
  • Real estate listings — apply the property's coordinates to the full set of listing photos so they group correctly in catalog tools.
  • Field documentation — surveyors, inspectors, and journalists who shoot a sequence at one site and need consistent geotags on every frame.
  • Photographer deliverables — geotag a full session's selects before handing them to a client's catalog software.

What gets written into each file

The latitude and longitude you choose are converted from decimal degrees into the degrees-minutes-seconds format the EXIF GPS IFD expects, plus the corresponding N/S and E/W reference tags. Existing EXIF fields — camera model, lens, exposure, capture timestamp — are preserved exactly. The image pixels are not re-encoded, so there is no quality loss between input and output.

Frequently Asked Questions

How many photos can I geotag at once?

Up to three JPEGs per drop, with no cap on how many batches you run back to back. The browser handles each file in memory, so the practical limit is your device's RAM — most laptops and modern phones handle 12-megapixel batches without issue.

Can I set different coordinates for different photos in the same batch?

Yes. Each uploaded photo has its own coordinate field and can be tagged with its own latitude and longitude. If you want the same location for all of them, set the coordinates once on the first photo and reuse them — the values stay in the input boxes between files.

Are the original files modified?

No. The originals on your disk are never overwritten. Each tagged photo is offered as a new download; if you need to keep the originals untouched, just save the new files to a different folder.

What format does the tool support?

JPEG/JPG, because EXIF GPS metadata is part of the JPEG specification. RAW formats (.CR2, .NEF, .ARW, etc.) are not supported here — for those, a desktop app like Lightroom or ExifTool is the right choice.

Will the geotag survive being uploaded to social media?

It depends on the platform. Instagram, Facebook, and Twitter strip EXIF on upload. Most cloud storage services (Google Drive, Dropbox), email attachments, and direct file transfers preserve it. If your end use is a photo catalog, mapping app, or your own website, the geotag will travel with the file.

I just want to fix a wrong location, not add a missing one.

The geo tag editor page is set up specifically for that — it shows the existing GPS values, lets you reposition the marker, and overwrites the file with the corrected coordinates. If you're looking to fix location metadata, our tool ensures perfect coordinate alignment.
