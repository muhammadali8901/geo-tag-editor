Important note about Google Search Console:
Files in this repository do not go to Google Search Console automatically.

Google Search Console will show the website only after:
1. The latest version is deployed on the live domain
2. Google can crawl the live pages
3. The sitemap is submitted in Search Console
4. Google indexes the pages

Current Search Console situation:
- Verification tag is added
- Sitemap exists
- Website is technically prepared for indexing
- Actual indexing depends on Google crawl and Search Console submission
